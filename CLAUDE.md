# Intelligence Lair - Development Instructions

## Important Rules

**NEVER ask "should I run this?" or "do you want me to test this?" - JUST RUN IT.**

Always run, test, and verify code yourself. Don't ask for permission to execute.

## Commands

```bash
# Activate environment
cd ~/Intelligence-Lair-School-Shootings
source venv/bin/activate

# Run the server
python app.py
```

## Project Overview

School threat detection system with:
- Live RTSP camera feeds
- YOLOv8 weapon detection
- Real-time dashboard with alerts
- Mobile app for notifications (Phase 2)
